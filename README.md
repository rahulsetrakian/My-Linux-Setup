# My Personal Kali Setup 

Minimal, KDE Plasma configuration for Kali Linux.

## Installation

```bash
git clone https://github.com/rahulsetrakian/My-Linux-Setup
cd My-Linux-Setup
chmod +x install.sh
./install.sh 
```

## Requirements
- Fresh Kali Linux installation
- Internet connection
- sudo access

## Features
- Minimal and Fast
- Single Script

## Configuration
- Custom Vim Configuration
- Oh My Zsh with predefined plugins and theme
