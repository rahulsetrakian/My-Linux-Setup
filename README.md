# My Personal Kali Setup 

Minimal, KDE Plasma configuration for Kali Linux.

## Installation

```bash
git clone https://github.com/rahulsetrakian/My-Linux-Setup
cd My-Linux-Setup
chmod +x install.sh
./install.sh 
```

## Features
- Minimal and Fast
- Single Script

## Configuration
- Vim Configuration
- OhMyZsh
